# TWK-Rental-Inventory-System-Modeling-Suite-FDD-DFD-UML-
Systems analysis modelling of the Teton Whitewater Kayak rental business using FDD, context/system/lower-level DFDs, and UML use case + sequence diagrams. Covers reservations, rentals, returns, used-equipment sales, procurement and reporting, with documented assumptions, exception flows, and peer-feedback refinements.
